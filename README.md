# Atividade de Programação Orientada a Objetos (POO) - UEMA

Este repositório contém uma atividade prática de Programação Orientada a Objetos desenvolvida para o curso de Análise e Desenvolvimento de Sistemas da UEMA. O projeto demonstra conceitos fundamentais de POO em Java, incluindo classes, objetos, herança, polimorfismo e encapsulamento.

## Pré-requisitos

- Java JDK 17 ou superior instalado.
- Visual Studio Code com a extensão Java instalada.
- Git para clonar o repositório.

## Como Clonar e Executar Localmente

1. **Clone o repositório:**
    Abra o terminal e execute o comando:
    ```
    git clone https://github.com/andrepfdev/java.git
    ```

2. **Navegue para o diretório do projeto:**
    ```
    cd java
    ```

3. **Abra no VS Code:**
    ```
    code .
    ```

4. **Compile e execute:**
    - Abra o arquivo principal (por exemplo, `src/Main.java`).
    - Use o comando integrado do VS Code para compilar (Ctrl+Shift+P > "Java: Compile") ou execute via terminal:
      ```
      javac -d bin src/*.java
      java -cp bin Main
      java src/App
      ```

## Estrutura do Projeto

- `src/`: Código-fonte Java.
- `lib/`: Dependências (se aplicável).
- `bin/`: Arquivos compilados (gerados automaticamente).

Para mais detalhes sobre gerenciamento de dependências, consulte a [documentação do VS Code Java](https://github.com/microsoft/vscode-java-dependency#manage-dependencies).
